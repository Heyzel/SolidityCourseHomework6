# No Loss Lottery

Basic no loss lottery to buy tickets with ETH, DAI, USDC and USDT.

## Dependencies

Install dependencies with `yarn`

## Compiling

```
yarn build
```

## Testing

```
yarn test
yarn test:coverage
```

## Deploying
To deploy the first version use

```
yarn local:deploy
```

And for upgrade use

```
yarn upgrade
```
